Restaurant Rating Prediction System

End-to-end ML system predicting restaurant ratings using location, cuisine, and customer data with Flask deployment.

Problem Statement

Restaurants struggle to predict customer ratings based on location and cuisine factors. This project delivers a scalable ML solution with Flask API integration for real-time predictions.

Tools & Technologies

Python, Flask, Scikit-learn, Pandas, NumPy, Git/GitHub, HTML/CSS/JS

Your Role & Responsibilities

  • Designed and trained machine learning models
  • Built Flask API for real-time predictions
  • Developed responsive frontend interface
  • Managed project with Git/GitHub version control

System Architecture

Raw data → Preprocessing → Model Training → Flask API → Responsive Frontend → Real-time Predictions

Key Features

  • ML-powered rating prediction with high accuracy
  • Complete ML + Flask API integration
  • Auto-suggestion input fields
  • Production-ready deployment structure
  • Clear role mapping for Data Analyst / ML Engineer
