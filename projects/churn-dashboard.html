E-Commerce Customer Churn Analysis Dashboard

Power BI | Business Intelligence

Problem Statement

Customer churn directly impacts business revenue and long-term growth. The objective of this project was to analyze customer churn behavior, identify high-risk segments, and provide actionable insights to support data-driven retention strategies using an interactive Power BI dashboard.

Tools

Power BI DAX Data Modeling

Key Insights

  • Approximately 29% of customers have churned, indicating a significant retention challenge
  • USA contributes the highest number of churned customers
  • Churn rate remains stable across signup quarters, suggesting behavior-driven churn rather than seasonality
  • Customer segmentation enables focused retention analysis by region and time

Role Mapping

  • Data Analyst: Designed churn KPIs, built DAX measures, and delivered actionable insights
  • Python Developer: Applied structured analytical logic and metric-driven thinking
  • AI/ML: Framed churn analysis in a predictive-ready format for future modeling
