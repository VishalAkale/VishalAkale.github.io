Restaurant Map & Analytics Dashboard

Python & Geospatial Analytics

Problem Statement

Understanding restaurant distribution, cuisine popularity, and city-level trends is difficult using raw tabular data alone. This project focused on building an interactive geospatial analytics dashboard to visualize restaurant data and support data-driven insights across cities.

Tools

Python Flask Pandas NumPy Leaflet.js Plotly

Responsibilities

  • Cleaned and standardized restaurant datasets (UTF-8 normalization, missing values)
  • Implemented interactive city & cuisine filters
  • Built geospatial maps using Leaflet.js
  • Designed analytical visualizations for cuisine distribution and regional trends
  • Integrated backend data pipelines with frontend dashboard

Key Outcomes

  • Clear visualization of restaurant density across Indian cities
  • Identification of cuisine popularity by region
  • Improved exploratory analysis using interactive filters and maps
  • Strong hands-on experience in geospatial analytics

Role Mapping

  • Data Analyst: Regional trend analysis & visual insights
  • Python Developer: Data processing & backend logic
  • AI/ML: Spatial pattern recognition & analytical thinking
