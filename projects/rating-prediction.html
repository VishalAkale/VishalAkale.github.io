Restaurant Rating Prediction System

End-to-end machine learning system to predict restaurant ratings based on city and cuisine.

Project Overview

Developed a complete machine learning pipeline including data preprocessing, model training, and Flask-based deployment to predict restaurant ratings.

Key Responsibilities

  • Data cleaning and preprocessing using Pandas and NumPy
  • Feature encoding for city and cuisine
  • Training and evaluating ML models
  • Deploying the model using Flask

Technologies Used

Python Pandas NumPy Scikit-learn Flask
