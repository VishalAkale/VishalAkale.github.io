Restaurant Rating Prediction System

End-to-end machine learning system developed to predict restaurant ratings based on city and cuisine type.

Project Overview

This project focuses on building a complete machine learning pipeline to predict restaurant ratings using historical data. It covers data preprocessing, model training, and deployment through a Flask-based web application.

Key Highlights

  • Developed an end-to-end ML pipeline from raw data to deployment
  • Handled categorical feature encoding for city and cuisine
  • Integrated trained ML model with a Flask backend
  • Implemented input validation for reliable predictions
  • Designed a clean interface for real-time usage

Problem Statement

Restaurant ratings depend on multiple factors such as location and cuisine. Manual estimation of ratings is subjective and inconsistent. This project aims to automate the prediction process using machine learning to deliver consistent and data-driven results.

Approach & Methodology

  • Cleaned and preprocessed data using Pandas and NumPy
  • Encoded categorical features such as city and cuisine
  • Trained and evaluated machine learning models
  • Exported the trained model for inference
  • Built a Flask web application for real-time predictions

System Workflow

  • Data Collection → Raw restaurant dataset
  • Data Preprocessing → Cleaning & encoding
  • Model Training → Machine learning model development
  • Model Export → Saved for inference
  • Deployment → Flask-based web application

Technologies Used

Python Pandas NumPy Scikit-learn Flask HTML CSS

Outcome & Learnings

The system successfully predicts restaurant ratings based on user inputs. This project strengthened my understanding of machine learning workflows, feature engineering, and deploying ML models as production-ready web applications.
