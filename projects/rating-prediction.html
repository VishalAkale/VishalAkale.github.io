Restaurant Rating Prediction System

End-to-end machine learning system developed to predict restaurant ratings based on city and cuisine type.

Project Overview

This project focuses on building a complete machine learning pipeline to predict restaurant ratings using historical data, covering preprocessing, model training, and deployment via Flask.

Key Highlights

  • End-to-end ML pipeline implementation
  • Feature encoding for city and cuisine
  • Flask-based real-time prediction system
  • Input validation and clean UI

Approach & Methodology

  • Data collection and preprocessing
  • Model training and evaluation
  • Model export for inference
  • Deployment using Flask

Technologies Used

Python Pandas NumPy Scikit-learn Flask
